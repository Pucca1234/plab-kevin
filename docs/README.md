## 현재 운영 방향
- 루트 `README.md`는 실행과 운영 진입 문서로 유지합니다.
- 루트 `PRD.md`는 공통 PRD 진입 문서로 유지합니다.
- 상세 동작 규칙은 `MASTER_PRD.md`와 기능별 PRD에 기록합니다.
- 후속 작업은 `docs/todo/*`에 남겨 다음 작업자가 바로 이어받을 수 있게 합니다.
